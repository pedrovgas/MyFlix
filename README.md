# MyFlix

Um aplicativo demonstrativo que utiliza Svelte e Tailwind para ajudá-lo a acompanhar suas séries e outras produções, marcando temporadas e episódios onde você parou em um layout ultra minimalista.

## Tecnologias Utilizadas

- [Svelte](https://svelte.dev/): Framework JavaScript moderno conhecido por sua eficiência e simplicidade, que realiza a maior parte do trabalho pesado durante a fase de compilação.
- [Tailwind CSS](https://tailwindcss.com/): Estrutura de CSS utilitário que permite estilizar aplicativos de forma rápida e eficiente, utilizando uma ampla gama de classes utilitárias.
- [Vite](https://vitejs.dev/): Ferramenta de construção de aplicativos frontend que proporciona uma experiência de desenvolvimento rápida e eficiente, com tempos de compilação ultrarrápidos e suporte a hot module replacement (HMR).

## Persistência Local do MyFlix

No MyFlix, os dados são armazenados localmente no navegador do usuário usando o armazenamento local (localStorage). Isso permite que suas séries e episódios marcados sejam retomados a qualquer momento, mesmo após fechar o navegador ou reiniciar o dispositivo.
